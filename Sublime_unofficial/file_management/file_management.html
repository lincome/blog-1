文件导航和文件管理¶

跳转到任何地方¶

跳转到任何地方可以让你迅速浏览文件。打开它通过Ctrl+P. 当你在输入区域键入时，打开文件和打开目录的名字将会被搜索，然后显示最匹配结果的预览。 预览是暂时的,也就是说, 它不会成为实际有效的缓冲区，直到你执行一些操作。当你按 Esc退出暂时视图。在其他情况你也会发现暂时视图。它们就像幽灵之类的东西。

跳转到任何地方不会徒有虚名--还有比它更好的定位文件方式:

要进行一次模糊搜索, 添加# 然后继续输入，像这样:

island#treasure

这会让Sublime Text模糊搜索treasure在文件名匹配island中。按Ctrl+; 将会打开跳至任何地方并且为你输入了#。

还有更多:

要搜素符号在激活缓冲区里,请按Ctrl+R。操作符 @就像上面解释的一样的作用。

要调至某一行号, 按Ctrl+G.操作符:就像上面解释的一样的作用。

搜索符号只会在定义过待搜索符号的文件中起作用。

侧边栏¶

侧边栏为您提供了您的项目的概述。添加到侧边栏的文件和文件夹可以在跳至任何地方和项目范围中使用。 项目和侧边栏是密切相关的。总有一个开放的项目，无论是显式或隐式的。

要打开或关闭侧边栏,按Ctrl+K, Ctrl+B。

可以通过上下箭头键导航侧边栏，但是你先要给它输入聚焦通过按Ctrl+0.要回到缓冲区的输入聚焦，按Esc. 或者，你也可以用鼠标相同的效果，但你会吗？

通过右键菜单侧边栏也提供了文件管理操作。

项目¶

项目组把你工作所需的文件和目录看作一个单元。 一旦你用适合你的方式建立项目，通过向其中添加目录，保存它给它一个名字。

要保存一个项目, 选择Project | Save Project As...。

项目中快速切换, 按Ctrl+Alt+P.

项目数据存储在有.sublime-project后缀的JSON文件里。 在有.sublime-project 文件的地方,你也会发现一个配套的.sublime-workspace文件。第二个是给Sublime Text用的，你不能自己编辑它。

项目文件可以定义项目独有的配置。更多见官方文档。

你可以通过命令行打开一个项目，只要把.sublime- project文件当做一个参数传入。
