安装¶

每个平台上的Sublime Text安装流程是不同的。

确定已经读了官网上的使用的条件。 Sublime Text 不是免费的。

32位还是64位?¶

选择64-bit版本如果你使用了一个64-bit操作系统, 否则还是32-bit版本吧。

在Windows , 如果有顾虑,选择32-bit版本. 当下64-bit 版本Windows可以运行32-bit软件。

在Linux 终端里运行以下命令来检查你的系统类型:

uname -m

对于OS X, 你可以跳过这个选择: 只有一个OS X用的Sublime Text版本。

Windows¶

便携还是非便携?¶

Sublime Text 安装到Windows有两种形式: 正常和便携.。如果你需要便携式安装，你或许已经知道了。否则，用正常安装吧。

正常安装 数据分开存放在单独的2个目录里: 适当的安装目录,和data 目录. 这些概念将在后面的指南中解释。正常安装也会集成Sublime Text到Windows 右键菜单。

便携安装 将保留所有Sublime Text运行需要的文件在一个单独的目录。 你可以移除这个目录但是编辑仍然可以工作。

如何安装正常版本的Sublime Text¶

下载安装包, 双击它按照屏幕上的说明去做。

如何安装便携版的Sublime Text¶

下砸压缩包然后解压它到你指定的一个目录。你会发现sublime_text.exe 执行程序在那目录里。

OS X¶

下载并打开.dmg 文件, 并且拖动Sublime Text 2包到Applications 目录。

Linux¶

你可以下载安装包并手动解压。相反你也可以使用以下命令:

cd /path/to/your/apps
wget http://url/to/sublime.tar.bz2
tar vxjf sublime.tar.bz2

如果你需要,为了方便你也可以创建一个执行文件的符号链接:

sudo ln -s /path/to/sublime_text /usr/bin/subl

活的危险点...或不¶

Sublime Text有三个发行渠道:

  • 稳定版 (默认的)
  • 开发版
  • 每夜版

如果你在做一个NASA项目或者有很短的期限,继续使用稳定版停止向下读。稳定版相比其他版本对于每天都用的人更好的测试了并且更可靠。他们大约每个月出一个稳定版。 大多数用户只想使用稳定版。

开发版 和 每夜版渠道是不稳定的，这意味着他们中发布的编译版极有可能包含bugs并且工作起来不可靠。 他们被更新的比稳定版更频繁。

开发版每个人都能使用。平均地，他们被每个月发布2次。尽管还没有准备好用于日常使用,他们以多数不间断方式展现新的特性。

最后， 每夜构建版处于最前沿，随着频繁的更新而带来频繁的不同严重性程度的问题。尝试他们很有趣，但是那么做也是在冒险。每夜构建版只给注册用户使用。
