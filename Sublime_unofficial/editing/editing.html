编辑¶

概述¶

Sublime Text充满了编辑特性。本主题只是说明了表面上可能的。

多重选择¶

多重选择使你高效地改变你的文字。任何关于多重选择的赞美是轻描淡写。这是原由:

选中一段文字然后按Ctrl + D 来添加更多实例。如果 你想跳过当前实例, 按Ctrl + K, Ctrl + D.

如果你多选了, 按Ctrl + U来撤销当前实例的选择。

多重选择转到多行¶

Ctrl + L扩展选区到该行的末尾。 Ctrl + Shift + L 分个选区到行。

你可以赋值多行选中的行到一个单独的缓冲区，在那儿编辑他们，再选中多行编辑过的内容粘贴回第一个缓冲区。

其他选择文本的方式¶

列表很长;所有可用选项都可以在Selection下找到。举一些列子:

  • 选择子单词 (Alt + Shift + <arrow>)
  • 选择整对括号 (Ctrl + Shift + M)
  • 选择到下个缩进的内容(Ctrl + Shift + J)
  • 扩大选择范围 (Ctrl + Shift + Space)

交换¶

想要交换2个字母或者更好点2个单词? 用下面的试试 Ctrl + T.

还有更多，更多...¶

编辑, 选区, 查找 和 跳转到菜单是寻找处理编辑工具的好去处。也许只用到一些，但是其余的当你需要的时候仍然可用。
