查找和替换 - 单文件¶

搜索¶

要打开缓冲区的搜索面板 , 按Ctrl + F. 在搜索面版上的选项和搜索行文可以被键盘控制:

切换正则表达式 Alt + R
切换大小写敏感 Alt + C
切换精确匹配 Alt + W
查找下一个 Enter
查找先前的 Shift + Enter
查找所有的 Alt + Enter

渐进式搜索¶

The 渐进式搜索面板可以被唤醒通过Ctrl + I. The only 和普通搜索面板不同的是只有在键入回车键时才开始搜索Enter key: 在渐进式搜索，它选择下一个缓冲区匹配项后会隐藏搜索面板。在渐进式搜索和普通搜索面板之间的选择是一个见仁见智的问题。

替换文本¶

你可以打开替换面板通过Ctrl + H.

替换所有: Ctrl + Alt + Enter

提示¶

其他在缓冲区搜索的方式¶

跳至任何地方通过提供操作符# 在当前缓冲区内搜索。搜索的内容是#操作符后面的。

其他在缓冲区搜索的快捷键¶

这些快捷键在搜索面板隐藏时有效。

用最近搜索的词向前搜索 F3
用最近搜索的词向后搜索 Shift + F3
用最近搜索的词全文搜索 Alt + F3

多行搜索¶

你可以输入多行的搜索模式。腰间如一个换行符,按 Ctrl + Enter 在搜索面板中。注意搜索面板是可以缩放的。
