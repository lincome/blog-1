搜索和替换 - 多个文件¶

搜索¶

要打开在多文件中搜索, 按 Ctrl + Shift + F. 你可以使用键盘控制搜索面板和搜索行为:

切换正则表达式 Alt + R
切换大小写敏感 Alt + C
切换精确搜索 Alt + W
搜索下一个 Enter

搜索范围¶

哪里字段在搜索面板中决定了哪儿去搜索。你可以定义搜索的范围在几个方式上:

  • 添加单个目录(Unix类型的路径, 甚至Windows)
  • 添加/排除的文件基于一个模式
  • 添加符号位置(<open folders>, <open files>)

你可以结合这些过滤器通过逗号分割它们,举个例子:

/C/Users/Joe/Top Secret,-*.html,<open files>

按...搜索面板中的按钮来显示一个包含这些按钮的菜单。

结果格式化¶

在搜索面板中你可以发现下面选项来定制搜索结果的格式:

  • 在单独视图/输出面板
  • 显示上下文

结果间导航¶

如果找到多个匹配项，你可以使用快捷键线性移动:

下个匹配 F4
前一个匹配 Shift + F4
